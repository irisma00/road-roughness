### Prerequisites

Install required packages

  ```sh
  npm install 
  ```

### Run Program

Regular: Start the program

  ```sh
  npm start
  ```

Underdevelopment: Automatically restarting the node application when file changes in the directory are detected

  ```sh
  nodemon app.js 
  ```
